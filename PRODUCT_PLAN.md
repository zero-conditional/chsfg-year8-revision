# Product Plan

## Problem

Many revision resources are either:

- too fragmented
- too dry
- too expensive
- or too awkward for non-technical families to use

## Users

Primary users:

- your daughter
- other KS3 students

Secondary users:

- parents helping with revision
- tutors looking for simple shared material

## Product Principles

1. No login required for the first version
2. Open instantly in a browser
3. Clear explanations before testing
4. Small wins and confidence-building
5. Mobile-friendly for quick practice

## MVP

### Content

- 1 subject to start, ideally Maths
- 3 topics
- 1 revision page per topic
- 5 to 10 quiz questions per topic

### Product

- home page
- subject page
- topic page
- quiz interactions
- printable layout

## Delivery Phases

### Phase 1

Prove the concept with a very small site.

### Phase 2

Add more topics and subjects.

### Phase 3

Add progress and personalization.

## Technical Recommendation

Use a static-first site, likely with a framework such as Next.js, Astro, or even plain HTML/CSS if we want maximum simplicity.

My recommendation for this repo:

- Next.js if you want room to grow
- Astro if you want content-first simplicity

## Success Criteria

The first version is successful if:

- a child can use it without explanation
- a parent can send a link and it just works
- one topic is genuinely more helpful than a worksheet alone

## Immediate Build Target

Create:

1. a landing page explaining the project
2. a Maths page
3. a single topic page such as Fractions
4. a short interactive quiz
