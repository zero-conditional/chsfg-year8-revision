Exam timetable

A simpler view of the Year 8 exam schedule

The timetable is split by teaching group and class code. This page lays it out in a way that is easier to scan.

Step 1

Find the right column

The letters C, H, S, L, E, U, and R are the main teaching groups. Use the column that matches the right group first.

  • C / A1
  • H / A2
  • S / A3
  • L / A4
  • E / B1
  • U / B2
  • R / B3

Step 2

Use the A and B core subject codes

For English, Maths, and Science, A and B refer to the timetable side, and the number shows the set within that side.

  • Letters = teaching groups
  • A or B = timetable side for core subjects
  • Number = set within that side
  • Core subjects follow the code shown in the column heading

Step 3

Remember the GL papers

The first three exam dates are GL assessments for the whole year group.

  • Tuesday 5 May 2026: GL English
  • Wednesday 6 May 2026: GL Maths
  • Thursday 7 May 2026: GL Science

How to use it

Use it to plan revision

Start with the next exam coming up, then work forward rather than trying to revise every subject at once.

Important note

Core subjects use the A/B side and set number

For English, Maths, and Science, follow the column with the correct core subject code: A1, A2, A3, A4, B1, B2, or B3.

Important note

GL assessments are together

The GL assessments take place in the Main Hall or Conference Suite. Those happen first, before the subject-specific papers.
